# Nodejs MYSQL REST API

### Installation

```
git clone https://github.com/fazt/nodejs-mysql-restapi
cd nodejs-mysql-restapi
docker-compose up
npm install
npm run dev
```
### Description
REST API (MVC) to interact with a DB in MySQL. It is about employees, their name and salary.

### Endpoints
```
GET -> /api/employee
GET -> /employees/:id
POST -> /api/employee
DELETE -> /employees/:id
PATCH -> /employees/:id
```
